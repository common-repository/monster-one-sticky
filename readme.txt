== Installation ==

1. Upload 'monster-onesticky' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How it works? =
Easy. Instal and after publish post all older post lost "Sticky" status. Also works with deferred posts.

= Как это работает? =
Легко. Установи и после публикации поста все остальные потеряют "Липкий" статус. Также работает с отложенными постами.
